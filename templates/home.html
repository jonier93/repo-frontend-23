Aplicación Web

Es el segundo lenguaje más básico para crear páginas web. El primero es HTML, con el cual se define el contenido del sitio web; mientras que con CSS se define la presentación y el diseño de este, es decir, CSS permite separar el contenido de HTML de la representación visual del sitio. En resumen, mediante este lenguaje se puede modificar los elementos de una página web, tal como la posición, forma, espaciados, colores, entre otros
